# Frontend Mentor - Social links profile

![Design preview for the Social links profile coding challenge](./preview.jpg)



 Social Links Profile

 Description
Ce projet est une solution au challenge "Social Links Profile" de Frontend Mentor. L'objectif est de créer une page de profil qui affiche les liens vers les réseaux sociaux de l'utilisateur. Ce projet a été réalisé en utilisant uniquement HTML et CSS.

 Fonctionnalités
- Profil Utilisateur :Affiche une image de profil et le nom de l'utilisateur.
- Liens de Réseaux Sociaux : Liste des icônes des réseaux sociaux avec des liens correspondants.
- Design Responsive : La mise en page s'adapte aux différentes tailles d'écran, des mobiles aux ordinateurs de bureau.


 Technologies Utilisées
- HTML5 : Pour structurer le contenu de la page.
- CSS3 : Pour styliser la page, y compris l'utilisation de Flexbox pour la mise en page et des transitions pour les animations.
